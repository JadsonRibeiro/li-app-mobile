# LI APP Mobile

<p> App to manage a Loja Integrada store.</p>

## Utilities

- Update stock (unique variant or all)
- Update price
- Apply discount on products
- Add category
- List brands, grades and variantios

## Technologies

- React Native
- Expo
- TypeScript
- Local Storage
- Navigation (Stack and Tab)

## Usage

```bash
# Clone the repository
$ git clone https://github.com/JadsonRibeiro/https://github.com/JadsonRibeiro/li-app-mobile.git

# Access root directory
$ cd li-app-mobile

# Install dependencies
$ yarn 

# Initalize 
$ expo start
```
